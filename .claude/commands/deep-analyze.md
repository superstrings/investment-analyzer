# 深度分析命令

对指定市场或股票进行全面深度分析，基于 V12 框架 + V2 估值模型。

## 参数

$ARGUMENTS - 市场代码 (HK/US/A) 或股票代码 (如 HK.00700)

## 核心原则

> 止损优先，估值先行，周期顺势，量价确认

## 执行步骤

### 1. 解析参数

判断参数类型：
- `HK` / `US` / `A` - 批量分析该市场
- `HK.00700` 格式 - 单只股票分析
- 无参数 - 分析所有市场

### 2. 执行分析

#### 单只股票分析

```bash
python main.py deep-analyze -u dyson -c $ARGUMENTS
```

#### 市场批量分析

```bash
# 港股
python main.py deep-analyze -u dyson -m HK --save

# 美股
python main.py deep-analyze -u dyson -m US --save

# A股
python main.py deep-analyze -u dyson -m A --save
```

### 3. 分析流程（每只股票）

#### Step 1: 估值筛选（V2 估值模型）

根据股票类型选择指标路径：

**成长股（AI/半导体/新能源）** → Forward PE + PEG
- Forward PE 与行业均值和历史均值对比
- PEG = Forward PE ÷ 预期EPS增速(%)
- PEG < 0.75 显著低估 / 0.75-1.0 值得买入 / 1.0-1.5 合理 / 1.5-2.0 偏贵 / >2.0 回避
- 仅对增速>10%的标的使用PEG

**周期/价值股（矿业/化工/银行）** → PB-ROE + PB/ROE
- PB/ROE = PB ÷ ROE(%)
- PB/ROE < 0.10 深度低估 / 0.10-0.15 性价比优秀 / 0.15-0.20 合理 / 0.20-0.30 偏贵 / >0.30 回避
- 仅在同行业内比较有意义
- 周期底部不看PE

**估值筛选不通过** → 标记不通过，不进入技术分析，直接建议"观望"

#### Step 2: 日K技术评分（满分12分）

**OBV评分（满分50→折算6分）**

| 维度 | 满分 | 关键判断 |
|------|------|---------|
| OBV趋势方向 | 15 | 上升+15/走平+8/下降+0 |
| 量价配合 | 15 | 价涨量增+15/背离+5/价涨量缩+0 |
| OBV vs 价格背离 | 10 | 正背离+10/无+5/负背离+0 |
| 近期量能变化 | 10 | 放量突破+10/缩量整理+6/放量下跌+0 |

**VCP评分（满分50→折算6分）**

| 维度 | 满分 | 关键判断 |
|------|------|---------|
| 收缩形态 | 15 | 明确收缩+15/初步+8/无+0 |
| 均线排列 | 15 | 多头排列+15/缠绕+8/空头+0 |
| 突破信号 | 10 | 放量突破+10/临近+5/无+0 |
| 回调幅度 | 10 | <10%+10/<20%+6/>20%+2 |

**周K修正**：周K强势+0.5~1分，弱势-0.5~1分

#### Step 3: 60分钟量价转换评分（满分10分）

| 评分项 | 满分 | 标准 |
|--------|------|------|
| 量能变化 | 4 | 底部放量/顶部缩量+4 |
| 转折确认 | 3 | K线形态反转+3 |
| OBV方向 | 3 | OBV转向确认+3 |

≥8分立即入场 / 6-7分可入场 / <6分不入场

### 4. 信号判定

| 总评分 | 信号 | 操作 |
|--------|------|------|
| ≥9 | 极强 | 加仓 |
| 7-9 | 强势 | 持有/建仓 |
| 5-7 | 中性 | 观望 |
| 3-5 | 转弱 | 减仓 |
| <3 | 弱势 | 止损 |

## 输出格式

- 每只股票独立输出分析
- Web Search：每只股票最多1次搜索（合并关键词）
- 估值筛选不通过→跳过技术评分，直接给结论
- 不输出评分过程的解释文字，直接给评分表格


## 价值陷阱识别

低PE + 低ROE + 高PB/ROE 组合是典型价值陷阱信号，需排查：
- ROE是否持续下滑（而非周期性低点）
- 行业是否面临结构性衰退
- 是否存在重大资产减值风险

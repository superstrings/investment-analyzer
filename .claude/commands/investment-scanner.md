# 全市场投资扫描器

融合七大投资体系的量化扫描框架，对全市场（HK/US/A）进行系统性扫描，输出：市场阶段判断、资产配置建议、精选标的池、交易时机信号。

**理论基础**:
- 史文森《非凡的成功》→ 资产配置层
- 欧奈尔 CANSLIM → 基本面选股
- 威可夫四阶段 → 市场/个股周期定位
- 马克·米勒维尼 SEPA → 趋势模板筛选
- 利弗莫尔 → 关键枢轴点 + 金字塔加仓
- 陈江挺《炒股的智慧》→ 风控与仓位
- 简放 → 交易纪律与心理

## 参数

$ARGUMENTS - 扫描模式：
- `full` — 全量扫描（市场阶段+配置+选股+信号），约需10-15分钟
- `scan` — 仅选股扫描（跳过配置分析）
- `phase` — 仅市场阶段判断
- `rebalance` — 仅资产配置再平衡检查
- 具体代码（如 `HK.00700,US.NVDA`）— 对指定股票做七维评分

默认: `full`

## 第一层：市场阶段判断（威可夫四阶段）

### 1.1 获取市场基准数据

同步各市场基准指数近 120 日 K 线：

```bash
python main.py sync klines -c "HK.800000" --days 120   # 恒生指数
python main.py sync klines -c "US.SPY" --days 120       # 标普500
python main.py sync klines -c "SH.000300" --days 120    # 沪深300
```

> 如基准不在 DB 中，通过 Web Search 获取近期走势数据

### 1.2 威可夫阶段量化判定

对每个市场，计算以下指标并判定所处阶段：

**阶段判定矩阵**:

| 指标 | Stage 1 蓄力 | Stage 2 上升 | Stage 3 派发 | Stage 4 下跌 |
|------|-------------|-------------|-------------|-------------|
| 价格 vs MA200 | 在下方但止跌 | 在上方且上升 | 在上方但走平 | 跌破且下行 |
| MA50 vs MA200 | 即将金叉 | 金叉且扩散 | 收窄或死叉中 | 死叉且扩散 |
| 成交量特征 | 下跌缩量+偶放量 | 上涨放量+回调缩量 | 上涨缩量+下跌放量 | 放量下跌 |
| OBV 方向 | 底部走平/翘头 | 持续向上 | 顶部走平/拐头 | 持续下行 |
| 波动率(ATR) | 收窄中 | 适中/扩大 | 高位震荡 | 急剧扩大 |
| 新高新低比 | 新低减少 | 新高增多 | 新高减少 | 新低激增 |

**量化评分（每项 0-3 分，满分 18）**:
- Stage 1 得分 > Stage 2 → 蓄力期
- Stage 2 得分最高 → 上升期（**唯一适合做多的阶段**）
- Stage 3 得分最高 → 派发期（减仓/对冲）
- Stage 4 得分最高 → 下跌期（空仓/做空）

**阶段内细分**:
- Stage 2 Early（刚突破 Stage 1）→ 🟢 最佳建仓期
- Stage 2 Mid（趋势确立）→ 🟢 持仓/加仓期
- Stage 2 Late（涨幅已大）→ 🟡 持仓但不加仓
- 阶段转换期（模糊）→ 🟡 轻仓观望

### 1.3 市场情绪温度计

通过 Web Search 获取当前市场情绪指标：
- 搜索: `"VIX" OR "恐慌指数" 当前数值 2026`
- 搜索: `"A股 成交额 万亿" OR "两融余额" 最新`
- 搜索: `"港股 南向资金 北向资金" 最新 流入 流出`

**情绪量化**（0-100）:

| 情绪 | 分值 | 操作含义 |
|------|------|---------|
| 极度恐慌 | 0-20 | 🟢 贪婪时刻（逆向加仓） |
| 恐慌 | 20-35 | 🟢 左侧布局 |
| 偏冷 | 35-50 | 🟡 正常运行 |
| 中性 | 50-65 | 🟡 正常运行 |
| 偏热 | 65-80 | 🟡 谨慎，不追高 |
| 贪婪 | 80-90 | 🔴 开始减仓 |
| 极度贪婪 | 90-100 | 🔴 大幅减仓/对冲 |

## 第三层：全市场选股扫描

### 3.1 扫描范围

从以下来源获取候选池：
1. 当前持仓 + 关注列表（数据库）
2. 各市场热门/强势股（Web Search，每市场1次）
   - `"港股 今日涨幅榜 强势股 2026年4月"` 
   - `"A股 涨停 强势股 连板 2026年4月"`
   - `"US stocks breaking out new highs April 2026"`

### 3.2 第一道滤网：米勒维尼趋势模板（SEPA）

对每只候选股同步 K 线数据并检查 8 项条件：

```bash
python main.py sync klines -c "[代码]" --days 250
```

**趋势模板检查清单**（全部通过 = ✅，任一不满足 = ❌）:

| # | 条件 | 计算方式 |
|---|------|---------|
| T1 | 价格 > MA150 且 > MA200 | close > MA150 AND close > MA200 |
| T2 | MA150 > MA200 | MA150 > MA200 |
| T3 | MA200 至少上升1个月 | MA200 连续20日斜率 > 0 |
| T4 | MA50 > MA150 且 > MA200 | MA50 > MA150 > MA200 |
| T5 | 价格 > MA50 | close > MA50 |
| T6 | 价格距52周低点 > 25% | close > 52w_low × 1.25 |
| T7 | 价格距52周高点 < 25% | close > 52w_high × 0.75 |
| T8 | 相对强度 RS > 70 | 近6月涨幅在市场前30% |

**通过标准**: T1-T5 全部满足（必须）+ T6-T8 至少满足2项
**评分**: 每项通过得1分，满分8分。≥7分 = A级趋势，5-6分 = B级，<5分 = 淘汰

### 3.3 第二道滤网：CANSLIM 基本面（欧奈尔）

对通过趋势模板的股票，Web Search 获取基本面（每只最多1次搜索）：

搜索关键词: `"[名称] 2025 2026 营收 净利润 增速 毛利率 机构持仓 行业地位"`

**CANSLIM 七维评分**（每维 0-15 分，满分 105 → 归一化为 0-100）:

| 维度 | 15分（优秀） | 10分（良好） | 5分（及格） | 0分（不及格） |
|------|-------------|-------------|-------------|-------------|
| **C** 当季盈利 | EPS增速>50% | 25-50% | 15-25% | <15% 或亏损 |
| **A** 年度盈利 | 3年EPS CAGR>25% | 15-25% | 10-15% | <10% |
| **N** 新产品/新高 | 有革命性新品+新高 | 有新品或新高 | 行业有催化剂 | 无催化剂 |
| **S** 流通盘 | <5亿股+近期缩量 | 5-20亿+正常量 | 20-50亿 | >50亿大盘股 |
| **L** 领导者 | RS>90+行业龙头 | RS 70-90 | RS 50-70 | RS<50 |
| **I** 机构认可 | 顶级机构+增持 | 知名机构持仓 | 一般机构 | 无机构/减持 |
| **M** 市场方向 | Stage 2 + 情绪中性 | Stage 2 Late | Stage 1 | Stage 3/4 |

**通过标准**: 总分 ≥ 60/100 且 C+A ≥ 20（盈利质量门槛）

### 3.4 第三道滤网：V2 估值（防高估陷阱）

对通过 CANSLIM 的股票，执行 V2 估值：

- 成长股: PEG < 1.5（合理以下）
- 价值股: PB/ROE < 0.20（合理以下）
- 高估值门槛: PEG > 2.0 或 PB/ROE > 0.30 → 一票否决

**估值修正**:
- 市场处于 Stage 2 Early → PEG 阈值放宽至 2.0（牛市初期容忍更高估值）
- 市场处于 Stage 2 Late → PEG 阈值收紧至 1.2（后期严格估值）
- 市场处于 Stage 3/4 → PEG 阈值收紧至 1.0（防守）

## 第四层：交易时机与风控

### 4.1 V12 技术评分

对通过三道滤网的股票执行现有 V12 评分体系：
- OBV 评分 (0-50)
- VCP 评分 (0-50)
- 总分折算 /12

### 4.2 利弗莫尔枢轴点

识别关键价格枢轴（Pivotal Point）:

**自然回升枢轴**: 下跌趋势中首次有意义的反弹高点
- 计算: 近60日内最低点后的第一个显著高点（涨幅>10%且放量）

**自然回调枢轴**: 上升趋势中首次有意义的回调低点
- 计算: 近60日内最高点后的第一个显著低点（跌幅>8%且缩量）

**突破枢轴**: VCP 成熟后的突破价位
- 计算: VCP pivot price（已有模块）

**持续枢轴**: 上升途中的缩量整理平台
- 计算: 近20日波动率收窄至<5%的价格中枢

**交易规则（利弗莫尔原则）**:
1. 只在枢轴点交易，不在无枢轴时操作
2. 突破枢轴 = 首次建仓（1/3仓位）
3. 突破后回踩枢轴不破 = 加仓（1/3仓位）
4. 持续枢轴突破 = 最后加仓（1/3仓位）→ 金字塔完成
5. 跌破最近一个枢轴 = 全部止损

### 4.3 综合评分与分级

**七维雷达图评分**（每维 0-100，总分 0-100）:

| 维度 | 权重 | 来源 | 说明 |
|------|------|------|------|
| 价值 (Value) | 20% | V2估值 | PEG/PB-ROE归一化 |
| 成长 (Growth) | 15% | CANSLIM C+A | 盈利增速 |
| 趋势 (Trend) | 20% | SEPA模板 | 8项通过率×100 |
| 量价 (Vol-Price) | 15% | V12评分 | OBV+VCP归一化 |
| 动量 (Momentum) | 10% | RS + 近期涨幅 | 相对强度 |
| 时机 (Timing) | 10% | 枢轴点距离 | 距最近枢轴的%距离 |
| 安全 (Safety) | 10% | 止损空间+波动率 | ATR/价格比 |

**综合评分 = Σ(维度得分 × 权重)**

**分级标准**:

| 评级 | 分数 | 含义 | 操作 |
|------|------|------|------|
| ★★★★★ | ≥85 | 极品（年出现2-3次） | 重仓买入+金字塔 |
| ★★★★ | 70-84 | 优秀 | 标准仓位买入 |
| ★★★ | 55-69 | 良好 | 轻仓试探/加入观察 |
| ★★ | 40-54 | 一般 | 仅观察不操作 |
| ★ | <40 | 不合格 | 淘汰 |

### 4.4 仓位管理（陈江挺 + 利弗莫尔）

**单股仓位上限**:

| 评级 | 最大仓位 | 金字塔方式 |
|------|---------|-----------|
| ★★★★★ | 20% | 10% → 7% → 3% |
| ★★★★ | 15% | 8% → 5% → 2% |
| ★★★ | 8% | 5% → 3% |

**总仓位控制（基于市场阶段）**:

| 市场阶段 | 总权益仓位 | 单日最大加仓 | 持股数量 |
|---------|-----------|------------|---------|
| Stage 2 Early | 80-100% | 15% | 5-8只 |
| Stage 2 Mid | 60-80% | 10% | 4-6只 |
| Stage 2 Late | 40-60% | 5% | 3-5只 |
| Stage 1/3 | 20-40% | 5% | 2-3只 |
| Stage 4 | 0-20% | 0% | 0-1只 |

### 4.5 止损规则（铁律）

| 类型 | 止损线 | 触发后 |
|------|--------|--------|
| 初始止损 | 买入价 -7% ~ -10% | 无条件执行 |
| 枢轴止损 | 跌破最近枢轴点 | 全仓离场 |
| 移动止损 | 浮盈20%后，从最高点回撤8% | 锁定利润 |
| 时间止损 | 买入后3周内未启动 | 半仓离场 |
| 盈利保护 | 浮盈50%后，从最高点回撤15% | 全仓离场 |

**陈江挺三不原则**:
1. **不懂不做** — 无法说清买入逻辑的不买
2. **不确定不做** — 信号不明确时宁可错过
3. **亏损不加仓** — 永远不对亏损头寸加仓

## 注意事项

### 扫描效率
- 趋势模板是最高效的滤网，先淘汰再做基本面
- Web Search 合并关键词，每只股票最多1次
- 批量同步K线时注意 Futu API 频率限制

### 框架优先级（冲突时的决策顺序）
1. **市场阶段 > 个股分析** — Stage 4 不做多，再好的股票也不买
2. **止损纪律 > 一切判断** — 触发止损无条件执行
3. **估值 > 技术** — V2 不通过，V12 再高也不买（价值陷阱例外）
4. **趋势 > 估值** — 便宜但 Stage 4 的股票是价值陷阱
5. **仓位纪律 > 信心** — 再看好也不超仓

### 适用场景
- 周末全量扫描（full 模式），制定下周计划
- 盘前快速扫描（scan 模式），确认当日关注
- 重大事件后（phase 模式），重新评估市场阶段
- 每月初（rebalance 模式），检查资产配置

### 复利曲线保护
- **年化目标 15-25%**，不追求暴利
- **最大回撤控制 -15%**，超过即大幅减仓
- **交易频率适中**，月均4-8次操作，避免过度交易
- **盈亏比 > 2:1**，每笔交易预期收益至少是风险的2倍
- **胜率 > 45%** + 盈亏比 > 2:1 = 正期望值系统

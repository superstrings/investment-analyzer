# 新股扫描命令

扫描近期上市新股，筛选有竞争力+技术面可交易的标的，用于短期趋势交易和长期跟踪。

## 参数

$ARGUMENTS - 市场（HK/US/A，默认全部）或具体代码（如 HK.06656）

---

## 执行步骤

### 1. 获取近期新股列表

通过 Web Search 获取最近 3 个月内上市的新股：

- 搜索关键词：`"[市场] 新股 IPO 上市 2026" site:futunn.com OR site:eastmoney.com OR site:stockstar.com`
- HK: 港股新股
- US: 美股IPO
- A: A股/北交所新股

每个市场获取最近10-15只新股的代码、名称、上市日期、招股价、当前价。

### 2. 基本面快筛（竞争力评估）

对每只新股进行快速基本面筛查（每只最多1次搜索）：

**必须满足（一票否决）：**
- [ ] 已盈利（或即将盈利，亏损收窄>50%）
- [ ] 营收增速 > 20%
- [ ] 毛利率 > 行业平均

**竞争力评分（满分12分，≥8分通过）：**

| 维度 | 3分 | 2分 | 1分 | 0分 |
|------|-----|-----|-----|-----|
| **市场地位** | 细分龙头/份额>15% | 前5名/份额5-15% | 前10名 | 无明显地位 |
| **增速** | 营收>50% YoY | 30-50% | 20-30% | <20% |
| **毛利率** | >40% | 25-40% | 15-25% | <15% |
| **护城河** | 技术+品牌+渠道 | 2项优势 | 1项优势 | 无壁垒 |

### 3. 技术面筛查

同步新股K线数据（上市以来），评估技术面：

```bash
python main.py sync klines -c "[代码]" --days 90
```

**技术面检查项：**

- [ ] **上市后走势**：破发回升 ✅ / 持续走强 ✅ / 持续破发 ❌
- [ ] **成交量趋势**：上市热度消退后仍有量 ✅ / 无量阴跌 ❌
- [ ] **OBV方向**：OBV走平或向上 ✅ / OBV持续下行 ❌
- [ ] **均线形态**：MA5>MA10>MA20多头排列 ✅ / 空头排列 ❌
- [ ] **价格位置**：在招股价上方 ✅ / 破发深度>20% ❌（除非有底背离）

**新股特殊规则：**
- 上市<20个交易日：无法计算MA20，仅看OBV和成交量趋势
- 上市<5个交易日：仅做基本面评估，技术面标注"数据不足"
- 新股前5天波动大属正常，不作为趋势判断依据

### 4. 趋势交易时机判断

对通过基本面+技术面的新股，判断当前是否有交易机会：

| 时机 | 条件 | 操作 |
|------|------|------|
| **上市热度回调企稳** | 首日暴涨后回调30-50% + OBV企稳 + 缩量 | 🟢 最佳买点 |
| **破发后底背离反转** | 破发后OBV底背离 + 放量收回招股价 | 🟢 反转买点 |
| **突破上市后整理区间** | 横盘2-4周后放量突破 + OBV新高 | 🟢 突破买点 |
| **上市仍在暴涨中** | 连续大涨 + 偏离招股价>50% | 🔴 不追，等回调 |
| **持续阴跌无企稳迹象** | OBV持续走低 + 无底背离 | 🔴 不碰 |

### 5. 输出结果

## 注意事项

- 新股数据有限（K线少），技术分析权重降低，基本面权重提高
- 上市首日/首周价格波动不作为趋势依据
- 港股新股暗盘价有参考意义，但不纳入技术分析
- 美股SPAC上市的不扫描（非传统IPO）
- A股注册制新股上市前5日无涨跌幅限制，波动大属正常
- 新股流动性可能不足，期权（如有）流动性更差，谨慎操作
- **核心原则：宁可错过，不可追高** — 新股最大风险是上市炒作后暴跌

## 分析框架

### 一、成长性评估（长期投资视角）

#### 1. 商业模式分析
- **收入结构**: MaaS(模型即服务) vs 应用收入 vs 硬件收入
- **客户类型**: ToB企业级 vs ToC消费级 vs 政府/国企
- **护城河**: 技术壁垒、数据壁垒、生态壁垒、客户粘性

#### 2. 关键成长指标
| 指标 | 说明 | 数据源 |
|------|------|--------|
| 营收增速(YoY) | >100%高速增长 | 财报/业绩预告 |
| 毛利率 | AI公司>50%为优秀 | 财报 |
| 收入留存率(NRR) | >120%说明客户持续加购 | 财报/招股书 |
| API调用量增速 | 反映模型使用活跃度 | 公司公告/新闻 |
| 研发投入占比 | AI公司通常>50% | 财报 |
| 亏损收窄趋势 | 关注经营亏损率是否改善 | 财报 |
| 现金储备/烧钱速度 | 能撑多久 | 财报 |

#### 3. 成长性排名
综合以上指标给出成长性评分（满分10分）：
- 营收增速(3分): >200%→3, >100%→2, >50%→1, <50%→0
- 毛利率(2分): >60%→2, >40%→1, <40%→0
- 商业模式(2分): 清晰可持续→2, 初步验证→1, 不明确→0
- 技术壁垒(2分): 行业领先→2, 有竞争力→1, 同质化→0
- 现金安全(1分): >2年→1, <2年→0

### 二、技术面分析（短期投机视角）

#### 1. V12 技术评分
运行 `python scripts/v12_scoring.py` 获取 OBV/VCP/周K 评分。

#### 2. 短期交易机会判断
| 信号 | 条件 | 操作 |
|------|------|------|
| 放量突破 | V12≥7 + 放量突破前高 | 正股或Call短线 |
| VCP蓄势 | VCP>30 + 缩量整理 | 等突破确认 |
| 事件催化 | 新模型发布/大客户签约/财报超预期 | 事件前布局 |
| 板块轮动 | AI板块资金流入 + 个股联动 | 跟随龙头 |

#### 3. 新股特殊处理
上市<60天的公司：
- V12评分不适用（数据不足）
- 用5分钟K线盘中分析替代（/intraday-analyze）
- 关注上市后的筹码结构（解禁日期、基石投资者）

### 三、估值对比（横向比较）

对这几家公司进行横向对比：

| 维度 | 指标 | 说明 |
|------|------|------|
| 市值 | 总市值(HKD) | 绝对规模 |
| PS(市销率) | 市值/年化收入 | AI公司核心估值指标 |
| 市值/研发 | 市值/年化研发投入 | 研发效率 |
| 用户规模 | MAU/API调用量 | 增长潜力 |
| 资金效率 | 收入/融资总额 | 资本使用效率 |

> 未盈利公司不适用PE/PB/ROE，使用PS(市销率)为核心估值指标
> PS < 行业中位数 = 相对低估，PS > 2倍中位数 = 相对高估

## 执行步骤

### 1. 数据收集
- 同步最新K线数据
- Web Search获取最新财报/业绩数据
- Web Search获取行业新闻和催化剂

### 2. 成长性评估
- 对每家公司评分（满分10分）
- 输出成长性排名

### 3. 技术面分析
- 运行V12评分（数据充足的公司）
- 对新股用近期K线形态判断

### 4. 估值横向对比
- 计算PS等估值指标
- 横向排名

### 5. 综合建议
输出两个维度的推荐：
- **长期投资**: 成长性最强的1-2家，适合定投/底仓
- **短期投机**: 技术面最强的1-2家，适合波段/期权

## 注意事项

- 大模型公司普遍**未盈利**，V2估值框架（PEG/PB-ROE）不适用
- 使用**PS(市销率)**作为核心估值指标
- 新股上市初期波动大，仓位严格控制（单只≤5%）
- AI板块情绪驱动强，技术面比基本面更重要（短期）
- 关注**解禁期**，基石投资者解禁可能造成抛压
- 这类公司属于**高风险高波动**，与V12框架中的"估值筛选"层冲突，操作属于**框架外投机**，需明确标注

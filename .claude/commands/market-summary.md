# 市场汇总命令

生成港股、美股、A股三市场综合分析汇总报告，基于 V12 框架 + V2 估值模型。

## 执行步骤

### 1. 同步最新数据

```bash
python main.py sync all -u dyson
```

### 2. 执行三市场深度分析

```bash
# 港股分析
python main.py deep-analyze -u dyson --market HK --batch

# 美股分析
python main.py deep-analyze -u dyson --market US --batch

# A股分析
python main.py deep-analyze -u dyson --market A --batch
```

### 3. 分析每只股票

对每只股票按 V12 框架执行：
1. **估值筛选**（V2 估值模型）：成长股用Forward PE+PEG，周期股用PB-ROE+PB/ROE
2. **日K技术评分**（12分制）：OBV(6分) + VCP(6分) + 周K修正
3. **信号判定**：≥9极强 / 7-9强势 / 5-7中性 / 3-5转弱 / <3弱势

### 4. 生成汇总报告

读取各市场分析报告，生成综合汇总：

```bash
# 查看汇总报告
cat reports/output/deep_analysis_summary_$(date +%Y-%m-%d).md
```

## 注意事项

- 建议每日盘前执行一次
- A股如无K线数据会显示待分析
- 报告保存在 reports/output/ 目录
- 基本面不通过，技术再好也不买
- 不追涨，等量价转换确认

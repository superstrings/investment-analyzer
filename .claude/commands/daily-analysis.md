# 每日分析命令

执行每日投资分析工作流（自动识别盘前/盘后），基于 V12 分析框架。

## 核心原则（V12 框架）

> 止损优先，估值先行，周期顺势，量价确认

### 决策优先级（严格按序执行）

1. **杠杆红线** → >2.0x强制减仓，>1.8x停止加仓
2. **止损执行** → 股票-10%，期权-30%，立即执行
3. **市场周期** → 牛(价>MA20周↑)/震荡(MA20周走平)/熊(价<MA20周↓)
4. **估值筛选** → 不通过则不进入技术分析
5. **日K评分** → ≥7分值得参与
6. **60分钟量价** → ≥6分可入场
7. **完整计划** → 无计划不执行

## 执行步骤

### 1. 判断当前时段

根据当前时间判断执行盘前还是盘后分析：
- **盘前** (开市前): 数据同步 + 估值筛选 + 技术评分 + 机会筛选
- **盘后** (收市后): 交易同步 + 持仓复盘 + 止损检查 + 风险检查

### 2. 盘前分析流程

```bash
# 1. 同步最新数据
python main.py sync all -u dyson

# 2. 港股深度分析
python main.py deep-analyze -u dyson --market HK --batch

# 3. 美股深度分析
python main.py deep-analyze -u dyson --market US --batch

# 4. A股深度分析 (如有数据)
python main.py deep-analyze -u dyson --market A --batch
```

分析完成后，对每只持仓/关注股票：
1. 先做估值筛选（V2 估值模型）
2. 通过后做日K技术评分（12分制）
3. 汇总信号和操作建议

### 3. 盘后分析流程

```bash
# 1. 同步交易记录
python main.py sync trades -u dyson

# 2. 同步最新K线
python main.py sync klines -u dyson

# 3. 查看持仓状态
python main.py account info -u dyson
```

分析完成后：
- 检查今日交易是否符合操作计划
- 更新持仓盈亏状态
- 检查止损触发情况（股票-10%，期权-30%）
- 检查杠杆率是否在安全范围
- 总结当日操作得失

### 4. 估值筛选规则（V2 估值模型）

| 股票类型 | 主用指标 | 通过标准 |
|---------|---------|---------|
| 成长股(AI/半导体) | Forward PE + PEG | PE < 行业均值且EPS增速>20%，PEG < 1.5 |
| 周期股(矿业/化工) | PB-ROE + PB/ROE | PB<行业均值 或 ROE>15%，PB/ROE < 0.20 |
| 重资产(银行/制造) | PB-ROE + PB/ROE | PB<1.5x且ROE>10% |

**筛选不通过** → 标记不通过，给出原因，不做技术评分，直接建议"观望/不参与"。

### 5. 日K技术评分（满分12分）

**OBV评分（满分50→折算6分）**：OBV趋势(15) + 量价配合(15) + OBV背离(10) + 近期量能(10)

**VCP评分（满分50→折算6分）**：收缩形态(15) + 均线排列(15) + 突破信号(10) + 回调幅度(10)

**周K修正**：周K强势+0.5~1分，弱势-0.5~1分

| 总评分 | 信号 | 操作 |
|--------|------|------|
| ≥9 | 极强 | 加仓 |
| 7-9 | 强势 | 持有 |
| 5-7 | 中性 | 观望 |
| 3-5 | 转弱 | 减仓 |
| <3 | 弱势 | 止损 |

## 输出格式

**Context管理**：
- 每只股票独立输出分析，不要累积到最后一次性输出
- Web Search：每只股票最多1次搜索
- 估值筛选不通过→跳过技术评分，直接给结论
- 不输出评分过程的解释文字，直接给评分表格

每只股票用以下紧凑格式：

```
## [股票代码] [名称] | ¥/$ [现价] ([涨跌幅])
估值: [Forward PE]x / PB [X]x / ROE [X]% → 通过 / 不通过
技术评分: [X]/12 (OBV [X]/50 + VCP [X]/50 + 周K修正[X])
信号: 买入 / 持有 / 减仓 / 卖出
关键位: 支撑 ¥XX(MA60) | 阻力 ¥XX(MA20)
操作: [具体建议，含止损价]
```

## 汇总报告格式

```
## 每日分析报告 - [日期]

### 时段: [盘前/盘后]
### 杠杆率: [X]x → [状态]

### 数据同步
- 持仓: [X] 条
- 交易: [X] 条
- K线: [X] 只股票

### [各股票独立分析...]

### 今日操作计划
[代码] [名称] | 方向: 买入/卖出
入场价: ¥XX | 仓位: X%
止损: ¥XX (-10%) | 目标: ¥XX
依据: 估值[通过/不通过] + 技术[X/12]
```

## 期权规则

- 仓位: 单个≤5%，同标的≤8%，总仓位≤15%
- 止盈止损: OCO#1(50%仓位) +30%/-30%，OCO#2(50%仓位) +50%/-30%
- 止损单绝不取消
- 最长持有5天，超7天强制平仓
- 入场条件: 日K≥7分，60分钟≥6分，IV Rank<70%，到期>45天
- 正股当日涨幅<3%（不追涨）

## 注意事项

- 盘前分析建议在开市前 30 分钟完成
- 盘后分析建议在收市后 1 小时内完成
- 周末执行时默认为盘后模式
- 基本面不通过，技术再好也不买
- 不追涨，等量价转换确认

# 持仓分析命令

快速分析当前持仓状况，基于 V12 框架风控规则。

## 执行步骤

### 1. 获取最新持仓数据

从数据库读取最新的持仓快照。

### 2. 分析维度

#### 杠杆检查（V12 最高优先级）

| 杠杆 | 状态 | 单股上限 | 操作 |
|------|------|---------|------|
| <1.5x | 正常 | <37.5% | 正常 |
| 1.5-1.8x | 警戒 | 37.5-45% | 停止加仓 |
| >1.8x | 危险 | - | 减仓 |
| >2.0x | 红线 | - | 强制减仓 |

#### 止损检查（V12 第二优先级）
- 股票亏损 ≥10% → 触发止损
- 期权亏损 ≥30% → 触发止损
- 列出所有需要止损的持仓

#### 仓位分析
- 各市场配比（港股/美股/A股）
- 单一股票集中度（上限 = 25% x 杠杆率）
- 现金比例

#### 期权持仓检查
- 单个期权仓位是否 ≤5%
- 同标的期权仓位是否 ≤8%
- 期权总仓位是否 ≤15%
- 持仓天数是否 >7天（超7天强制平仓）

#### 盈亏分析
- 持仓盈亏排名
- 总体盈亏
- 盈亏贡献度

### 3. 输出分析结果

## 风险预警规则（V12 对齐）

| 风险类型 | 预警条件 | 级别 |
|---------|---------|------|
| 杠杆红线 | 杠杆>2.0x | 紧急 |
| 杠杆危险 | 杠杆>1.8x | 严重 |
| 止损触发 | 股票亏损≥10% | 严重 |
| 期权止损 | 期权亏损≥30% | 严重 |
| 期权超时 | 期权持仓>7天 | 警告 |
| 单一集中 | 单股>25%x杠杆率 | 警告 |
| 高度集中 | 单股>30% | 严重 |
| 期权超仓 | 期权总仓>15% | 警告 |

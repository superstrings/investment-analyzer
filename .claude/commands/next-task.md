# 获取下一个任务命令

从 TASKS.md 选择并开始下一个任务。

## 执行步骤

1. **读取任务列表** - 读取 `TASKS.md`

2. **选择任务** - 按优先级顺序选择：
   - 优先选择 `in_progress` 状态的任务（继续未完成的工作）
   - 其次选择最高优先级的 `pending` 任务

3. **展示任务详情**
   ```
   ## 选中任务: [任务ID]

   **标题**: [任务标题]
   **优先级**: [P0/P1/P2/P3]
   **当前进度**: [X%]

   ### 子任务
   - [ ] 子任务1
   - [ ] 子任务2

   ### 相关文件
   - file1
   - file2

   ### 参考文档
   - doc1
   ```

4. **读取相关文档** - 自动读取任务相关的参考文档

5. **更新任务状态**
   - 将选中任务的 `status` 改为 `in_progress`
   - 更新 `TASKS.md`

6. **询问用户** - 确认是否开始此任务，或选择其他任务

## 选择策略

| 情况 | 行为 |
|-----|------|
| 有 in_progress 任务 | 继续该任务 |
| 多个 P0 任务 | 选择 progress 最高的（接近完成） |
| 无 P0 任务 | 选择最高优先级任务 |

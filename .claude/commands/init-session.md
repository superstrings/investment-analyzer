# 会话初始化命令

执行标准会话初始化流程，了解项目当前状态。

## 执行步骤

1. **确认工作目录**
   ```bash
   pwd
   ```

2. **查看最近进展** - 读取 `claude-progress.txt` 最后 50 行

3. **查看最近提交**
   ```bash
   git log --oneline -10
   ```

4. **查看当前任务** - 读取 `TASKS.md`

5. **检查是否有未提交的变更**
   ```bash
   git status
   ```

6. **汇报状态** - 向用户简要报告：
   - 最近完成的工作
   - 当前进行中的任务
   - 下一个建议任务

## 输出格式

```
## 会话初始化完成

### 最近进展
[最近工作摘要]

### 当前状态
- 未提交变更: [有/无]
- 进行中任务: [任务ID - 任务名]

### 建议下一步
[下一个 P0/P1 任务建议]
```

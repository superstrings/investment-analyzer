# 同步所有数据命令

快速同步所有投资数据（持仓、交易、关注列表、K线）。

## 执行步骤

### 1. 检查 FutuOpenD 状态

确认富途 OpenD 客户端已运行。

### 2. 执行全量同步

```bash
python main.py sync all -u dyson
```

### 3. 验证同步结果

检查同步日志，确认数据已正确写入。

### 4. 显示同步汇总

## 同步内容

| 数据类型 | 来源 | 说明 |
|---------|------|------|
| 持仓 | 富途 API | 当前持仓快照 |
| 交易 | 富途 API | 近365天交易记录 |
| 关注列表 | 富途 API | 自选股列表 |
| K线 (HK/US) | 富途 API | 近250天日K线 |
| K线 (A股) | akshare | 近250天日K线 |

## 输出格式

```
## 数据同步完成

### 同步时间
[时间]

### 同步结果
| 数据类型 | 状态 | 数量 |
|---------|------|------|
| 持仓 | ✅ | [X] 条 |
| 交易 | ✅ | [X] 条 |
| 关注列表 | ✅ | [X] 条 |
| K线 (HK) | ✅ | [X] 只 |
| K线 (US) | ✅ | [X] 只 |
| K线 (A) | ✅ | [X] 只 |

### 耗时
[X] 秒

### 数据更新
- 新增持仓: [X] 条
- 新增交易: [X] 条
- 更新K线: [X] 只股票
```

## 注意事项

- 需要 FutuOpenD 运行中
- A股K线使用 akshare，无需代理
- 首次同步可能耗时较长 (约 2-5 分钟)
- K线默认同步250天，支持技术分析

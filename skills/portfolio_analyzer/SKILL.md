# 投资组合分析师

## 能力描述
分析用户的投资组合，评估仓位配置、风险暴露、盈亏状况。

## 数据来源
- 数据库表: positions, accounts, account_snapshots
- 调用: python main.py report --user {user} --type portfolio

## 分析维度
1. 仓位分析
   - 各市场配比（港股/美股/A股）
   - 单一股票集中度
   - 现金比例

2. 盈亏分析
   - 持仓盈亏排名
   - 总体盈亏
   - 盈亏贡献度

3. 风险评估
   - 集中度风险（单一持仓>20%预警）
   - 杠杆率
   - 止损检查

## 输出格式
- 持仓明细表
- 仓位配比饼图
- 风险评分（1-10）
- 操作建议
